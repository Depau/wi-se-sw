## Prerequisites

install [Yarn](https://yarnpkg.com), and run: `yarn install`.

## Development

1. Start ttyd: `ttyd bash`
2. Start the dev server: `yarn run start`

## Publish

Run `yarn run build`, this will compile the inlined html to `../wi_se/html.py`.
